# WebOS 1 Desktop 🚀

A fully functional, browser-based operating system interface built from scratch. Designed as a customizable workspace with draggable glass-morphism windows, built-in apps, and direct links to Hackclub's ecosystem. Created for the Hackclub Stardance WebOS 1 mission.

**[Try it live!]** *(Open `index.html` in any web browser to get started)*

---

## 📸 Features

- **🖥️ Draggable Windows:** Drag windows by their title bar, click to bring them to the front (Z-index stacking).
- **📝 Notepad App:** A clean text area to jot down notes.
- **🧮 Calculator App:** A fully functional browser calculator.
- **📋 Task Hub:** Add and delete to-do items dynamically.
- **🔗 Quick Links:** Dedicated app windows for:
  - Hackclub / Stardance (`hackclub.com`)
  - Hackatime (`hackatime.hackclub.com`)
  - YouTube (`youtube.com`)
- **⚙️ Custom Settings:** Change the desktop background instantly using pre-set gradients or a custom color picker.
- **🔍 Global Search Bar:** Center-positioned search bar that opens a Google search in a new tab on Enter.
- **🕒 Live Clock:** Real-time digital clock positioned at the top right.

---

## 🛠️ Technologies Used

- **HTML5** – Semantic page structure.
- **CSS3** – Flexbox layouts, Glass-morphism, CSS Gradients, and hover effects.
- **Vanilla JavaScript** – Drag-and-drop logic, DOM manipulation, and app functionalities (Calculator, Clock, Task List, Settings).

---

## 🚀 Getting Started

1. Clone the repository or download the `.zip` file.
2. Make sure you have these three files in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Double-click `index.html` to open the WebOS in your browser.

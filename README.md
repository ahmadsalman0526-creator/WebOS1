WebOS 1 Desktop 🚀
This is an OS UI built using only the browser with no dependencies on anything else. The goal of creating this project was to design a very customizable work environment, which includes glass-morphism windows, applications, and fast access to Hackclub’s ecosystem. This project was done for Hackclub Stardance WebOS 1 challenge.
- Draggable Windows: The windows can be dragged by their title bars; when clicked, they come to the foreground (Z-index).
- Notepad App: A simple app where you can write down some notes.
- Calculator App: A functional browser-based calculator.
- Task Hub: Ability to create and delete tasks dynamically.
- Quick Links: Windows with predefined apps that will lead you to:
  - Hackclub / Stardance (hackclub.com)
  - Hackatime (hackatime.hackclub.com)
  - YouTube (youtube.com)
- Custom Settings: Ability to change your desktop background using pre-built gradients or choosing your own colors via color picker.
- Global Search Bar: A search bar located at the center of the screen; typing something and then pressing Enter will start searching Google on a new tab.
- Real Time Clock: A digital clock displayed at the top right corner of the screen.

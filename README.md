# Retro Desktop Interface

I built this retro desktop interface for the Hackclub Stardance mission. Most AI templates push the standard dark glassmorphism look, but I wanted to do something completely different. I decided to go with a classic 90s game console aesthetic instead. I swapped the boring teal background for a deep plum purple, used the "Press Start 2P" pixel font, and added sharp 3D bevel borders to make it look super retro.

## Features

- **OS Basics**: Windows that you can actually drag around, a working notepad, a calculator, and a task hub.
- **Playable Snake Game**: A fully working classic Snake game. You use the arrow keys to move and the spacebar to reset it.
- **Live Cat Generator**: It pulls random cat images from a public API in real-time.
- **DVD Screensaver**: The classic bouncing DVD logo that hits the corners.
- **Live Themes**: You can change the background color instantly with presets or a custom color picker.

## How I Built It

I broke the whole development process down into three devlogs:

- Devlog 1: Building the initial HTML skeleton.
- Devlog 2: Doing the custom CSS styling to get the retro vibe.
- Devlog 3: Writing the JavaScript logic to make all the apps actually work.

Everything is completely interactive and you can drag any window. There is no password or login required so anyone can just click the link and test it out right away!
